### 1. Clone the Repository

---

### 2. Backend Setup

cd backend  
npm install  

Create a .env file in the backend directory and add:

PORT=5000  
JWT_SECRET=your_jwt_secret  

Start the backend server:

npm run dev  

---

### 3. Frontend Setup

cd frontend  
npm install  
npm start  

The application will run at:

http://localhost:3000
